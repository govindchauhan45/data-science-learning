# 📊 Data Science Learning

This repository contains my practice and learning work in Data Science using Python.

The notebooks demonstrate examples and exercises using popular data science libraries.

## 🚀 Libraries Used

- NumPy → Numerical computing
- Pandas → Data manipulation and analysis
- Matplotlib → Data visualization
- Seaborn → Statistical visualization
- Jupyter Notebook → Interactive coding environment

## 📁 Project Structure

data-science-learning/

numpy/ → NumPy practice notebooks  
pandas/ → Pandas practice notebooks  
matplotlib/ → Visualization using Matplotlib  
seaborn/ → Statistical plots using Seaborn  

requirements.txt → Required Python libraries  
README.md → Project documentation  

## 📚 Topics Covered

### NumPy
- Array creation
- Mathematical operations
- Matrix operations

### Pandas
- DataFrames
- Data cleaning
- Data analysis

### Matplotlib
- Line plots
- Bar charts
- Histograms

### Seaborn
- Scatter plots
- Distribution plots
- Heatmaps

## ⚙️ Installation

Install the required libraries:

pip install -r requirements.txt

## ▶️ Running the Project

Open notebooks using Jupyter Notebook:

jupyter notebook

Then open any notebook file from the folders.
